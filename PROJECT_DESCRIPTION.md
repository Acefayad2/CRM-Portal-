# Pantheon - Project Description for AI Assistants

Pantheon is a Next.js 14 (App Router) web application for insurance and financial services agents. The tagline is "Protect Today. Grow Tomorrow." It serves as an internal hub where agents manage their clients, pipeline, appointments, and team workflows. Key features include: authentication (email/password and Google OAuth via Supabase), a dashboard with pipeline tracking (New, Contacted, Appt Set, Presented, Follow-Up), client management with list and board views, shared team calendars with availability, sales scripts and templates (presentations, cold calls, objection handling, follow-ups), training resources and modules (licensing, CFT, appointments), SMS notifications via Twilio, and team-based subscriptions where admins purchase a plan and share a team code for agents to join. The subscription tiers are Team, Business, and Enterprise. The tech stack is Next.js, TypeScript, Tailwind CSS, Supabase (auth + optional DB), Twilio (SMS), and Radix/shadcn UI components. Users sign up, optionally join a team with a code, see a subscription picker popup when entering the portal, and then access the main dashboard. The app uses Server Components where possible and Client Components for interactive UI (forms, modals, auth).
